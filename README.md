* How to use this algorithm:

# if your testcase want to be hand-modify, create a file in testcase folder and name it as
# e.g. testcase?in.txt, while ? is the id.
# Else you can use testcasegen.py first to generate 1000 random testcases out.
# Next, if you want to check a specific one, you can use 
python xor.py <inputfile> <outputfile>
notice that outputfile should be testcase?out.txt, ? is the same as input id.
# Or you can simply use py main.py after py testcasegen.py to gather 1000 outputs.

# All of the results are exact xor algorithm.
